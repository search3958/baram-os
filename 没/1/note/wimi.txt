1. **赤（RGB: 255, 0, 0）** → `idt_install()` の前に表示
   - IDT（割り込みディスクリプタテーブル）をインストールする直前
   - この色だけ表示されてクラッシュ = `idt_install()` でクラッシュしている

2. **黄（RGB: 255, 255, 0）** → `idt_install()` の後、`irq_install()` の前に表示
   - IDT のインストール成功
   - `irq_install()` でクラッシュしている可能性

3. **緑（RGB: 0, 255, 0）** → `irq_install()` の後、`mouse_install()` の前に表示
   - IRQ（割り込みリクエスト）のインストール成功
   - `mouse_install()` でクラッシュしている可能性

4. **シアン（RGB: 0, 255, 255）** → `mouse_install()` の後に表示
   - マウス初期化成功
   - メインループまで到達
